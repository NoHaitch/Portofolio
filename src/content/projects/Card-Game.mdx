Card Game based on OOP Java with JavaSwing UI and Maven.

In this projects, we fully explore all of the concepts of Object Oriented Programming (OOP) in Java. We also use Java Swing for the UI and Maven for the build tool.
We also use many design patterns such as Singleton, Factory, and Observer.
