Linier Algebra Calculator and Image Scaling using Java CLI.

Contains programs on Systems of Linear Equations (SPL), Determinants, Polynomial Interpolation, Multiple 
Linear Regression, and Bicubic Spline Interpolation. To solve SPL, use the Gauss Elimination Method, Gauss-Jordan, 
Inverse Matrix, and Cramer's Rule.
